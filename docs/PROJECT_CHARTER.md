# Project Phoenix 140 Charter

## Project Name
Phoenix 140 – Life Operating System

## Purpose
Phoenix 140 is a personal operating system for long-term asset building, cash flow management, real estate operations, investment decision-making, and health tracking.

## Core Philosophy
- Build long-term cash flow through dividends and real estate.
- Protect core assets while using non-core assets for strategic capital allocation.
- Treat health as a primary life asset.
- Make decisions based on data, not emotion.
- Preserve optionality for future investments.
- Build software that can grow for 10 years.

- Takeda Pharmaceutical
- JT

### Adjustable Holdings
- Shipping stocks

## Long-term Vision
Phoenix 140 should become Tomokazu's private AI CFO and life dashboard.
