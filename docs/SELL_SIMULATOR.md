# Sell Simulator

## Purpose

The Sell Simulator estimates:
- Gross sale proceeds
- Realized gain/loss
- Taxable gain
- Tax
- Net cash after tax
- Annual dividend reduction
- Shortfall or surplus against target cash

## Current Assumptions

- Taxable account
- Tax rate default: 20.315%
- Losses offset gains in the same simulation
- Transaction fees ignored
- NISA not yet modeled
- Foreign tax not yet modeled

## Planned Enhancements

- Auto-optimization to reach target net cash
- Preserve core holdings rule
- Sector preservation rule
- Dividend loss minimization
- Tax-loss harvesting assistant
- Multi-scenario comparison
