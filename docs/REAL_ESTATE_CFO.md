# Sprint 3 Real Estate CFO

## Purpose

Sprint 3 starts the real estate CFO layer.

The goal is to summarize:
- property count
- unit count
- occupancy
- monthly rent
- annual rent
- annual expenses
- NOI
- annual debt service
- cash flow after debt
- DSCR

## Current Design

Sprint 3.0 is read-only.

It reads existing database tables when available:
- properties
- units
- monthly cashflows
- loans
- repairs

It does not change the database schema.
