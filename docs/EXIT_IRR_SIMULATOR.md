# Sprint 3.4 Exit / IRR Simulator

## Purpose

Sprint 3.4 adds an exit return simulator for real estate investments.

It connects:
- initial cash required
- annual cash flow
- holding period
- loan balance at exit
- sale price
- selling costs
- simplified capital gain tax
- cumulative cash flow
- IRR

## New Page

```text
Exit / IRR Simulator
```

## Key Outputs

- remaining loan balance
- sale cash after debt and tax
- cumulative cash flow
- total recovery
- total profit
- equity multiple
- simple annual return
- IRR
- CFO judgement

## Default Scenario

The default scenario uses the oblige新田町 plan as a starting point.

## Simplifications

This simulator is intentionally simple.

It does not yet fully model:
- Japanese real estate capital gains tax rules
- land/building allocation
- depreciation recapture details
- acquisition tax
- registration and license tax
- income tax impact during holding period
- principal/interest split for tax accounting
- repairs and capex timing

## Future Enhancements

- depreciation and tax module
- land/building allocation
- refinance and sale comparison
