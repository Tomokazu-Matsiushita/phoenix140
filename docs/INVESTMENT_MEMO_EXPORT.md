# Sprint 4.3 Investment Memo / Decision Report Export

## Purpose

Sprint 4.3 adds exportable investment decision memos.

It converts the Integrated AI CFO final recommendation into a structured report.

## New Page

```text
Investment Memo / Decision Report Export
```

## Export Formats

- Markdown investment memo
- Scenario review CSV
- Decision factors CSV

## Memo Sections

- Final Recommendation
- Best Scenario Snapshot
- Decision Factors
- Acquisition Summary
- Exit / IRR Summary
- Liquidity & Safety
- Scenario Comparison
- Must Check Before Action
- CFO Note

## Current Design

This version exports Markdown and CSV.

PDF export is intentionally deferred to the next sprint.

## Future Enhancements

- PDF export
- HTML report
- saved memo history
- side-by-side memo comparison
- electronic approval workflow
