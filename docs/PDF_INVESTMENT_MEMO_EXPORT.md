# Sprint 4.4 PDF Investment Memo Export

## Purpose

Sprint 4.4 adds PDF export for the Phoenix 140 investment decision memo.

## New Page

```text
PDF Investment Memo Export
```

## Output

- A4 portrait PDF
- Japanese-capable ReportLab CID font

- Scenario comparison
- Must-check list
