# Sprint 3.1 Real Estate Model

## Purpose

Sprint 3.1 enriches the existing real estate data model.

It adds property-level operating metrics:

- current annual income
- full annual income
- current monthly income
- full monthly income
- occupied units
- vacant units
- fixed asset tax
- management fee
- cleaning fee
- insurance
- data quality note
